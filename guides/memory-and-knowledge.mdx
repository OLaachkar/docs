## Two kinds of memory

- **Short-term context**: the recent messages the model can see.
- **Long-term memory**: notes or facts you store to keep a consistent world.

## When to add knowledge

Add facts your verb should never forget. Examples:

- Names and roles in a story world.
- Rules for a game or community.
- Preferences you want the verb to remember.

<Tip>
  Store the important stuff once, then keep chat prompts short.
</Tip>

## Keep it clean

- Review old entries and remove outdated info.
- Avoid dumping full conversations into memory.
- Use clear labels and short sentences.

## Auto memory instructions

Use **Auto-Memory Instructions** to control what Verba saves automatically from chats.

### Where to set it

1. Open your verb in the dashboard.
2. Go to **Memory**.
3. Turn on **Auto Memory**.
4. Fill in **Auto-Memory Instructions**.
5. Save changes.

### What to write

- Focus on durable details: preferences, relationships, long-term goals, commitments.
- Exclude short-lived details: timestamps, one-off plans, channel metadata.
- Never store secrets, private links, passwords, or tokens.
- Keep memories concise and specific.

### Example instructions

```text
Store durable facts about user preferences, relationships, and commitments.
Skip temporary details like timestamps or one-off plans.
Never store secrets, passwords, tokens, or private URLs.
Write concise memories that are useful in future conversations.
```

<Tip>
  Auto memory is selective. If a conversation does not contain durable
  information, no new memory may be saved.
</Tip>
