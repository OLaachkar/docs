## Quick checks

- Refresh the page.
- Log out and back in.
- Try a different browser.

## Common issues

<AccordionGroup>
  <Accordion title="My verb is too generic">
    Lower the temperature, tighten the personality, and add examples.
  </Accordion>
  <Accordion title="Uploads are failing">
    Check file size and format. Stick to JPG or PNG and keep it under 10MB.
  </Accordion>
  <Accordion title="Web search is noisy">
    Turn it off unless you need current info. Lower top-p if it keeps drifting.
  </Accordion>
  <Accordion title="Slow responses">
    Use faster models, reduce context size, and avoid web search for casual use.
  </Accordion>
</AccordionGroup>
